# Unemployment Analysis with Python

## Objective

The objective of this project is to analyze unemployment trends using Python by performing data cleaning, exploratory data analysis (EDA), and visualization. The project also investigates the impact of the COVID-19 pandemic on unemployment rates and identifies important trends that can support economic and social policy decisions.

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Dataset

- **Dataset:** Unemployment Rate in India
- The dataset contains information such as:
  - Region
  - Date
  - Frequency
  - Estimated Unemployment Rate (%)
  - Estimated Employed
  - Estimated Labour Participation Rate (%)
  - Area (Urban/Rural)

---

## Project Workflow

1. Imported required libraries
2. Loaded the dataset
3. Cleaned and preprocessed the data
4. Performed Exploratory Data Analysis (EDA)
5. Created visualizations to identify trends
6. Compared unemployment before and during COVID-19
7. Generated insights and conclusions

---

## Visualizations

The project includes the following visualizations:

- Unemployment Rate Over Time
- State-wise Unemployment Rate
- Correlation Heatmap
- Distribution of Unemployment Rate (Histogram)
- Box Plot of Unemployment Rate
- COVID-19 Impact Comparison
- Top 10 States with Highest Average Unemployment
- Urban vs Rural Unemployment Comparison

---

## Key Insights

- Unemployment increased significantly during the COVID-19 pandemic.
- Different regions experienced different unemployment levels.
- Some states consistently recorded higher unemployment than others.
- Urban and rural unemployment patterns were different.
- The analysis highlights the economic impact of COVID-19 and the need for targeted employment policies.

---

## Conclusion

The analysis demonstrates that COVID-19 had a significant impact on unemployment rates across different regions. The visualizations reveal important trends and regional variations that can help policymakers understand employment challenges and design effective economic recovery strategies.
